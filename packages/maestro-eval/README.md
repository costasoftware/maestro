# maestro-eval (reserved)

This name is reserved for a future evaluation harness on top of [`@maestro/core`](https://www.npmjs.com/package/@maestro/core). Superseded by the live [`@maestro/evals`](https://www.npmjs.com/package/@maestro/evals) package; this placeholder will be deprecated. It would have shipped:

- Golden-path runner — fixture conversations with expected tool calls
- Injection / jailbreak / PII test suite
- Traffic replay against recorded production turns
- Promptfoo integration

See the [Maestro repo](https://github.com/costasoftware/maestro) for design status. Don't depend on this `0.0.0` placeholder — it has no runtime code.
