# create-maestro (reserved)

This name is reserved for a future project scaffolder following the `create-*` convention. Eventual usage:

```bash
npm create maestro@latest my-agent
```

Will bootstrap a Next.js (or other host) project with `@maestro/core` wired, port adapters stubbed, and example tools registered.

See the [Maestro repo](https://github.com/costasoftware/maestro) for design status. Don't depend on this `0.0.0` placeholder — it has no runtime code.
