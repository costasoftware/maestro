# maestro-mcp (reserved)

This name is reserved for a future Model Context Protocol **client** adapter on top of [`@maestro/core`](https://www.npmjs.com/package/@maestro/core). It will let the kernel **consume** external MCP servers as tools, complementing the MCP **server** adapter that already ships in `@maestro/core/adapters/mcp-server`.

See the [Maestro repo](https://github.com/costasoftware/maestro) for design status. Don't depend on this `0.0.0` placeholder — it has no runtime code.
